📝To-Do List App📝

Overview

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This app allows users to add, remove, and mark tasks as completed.

Features

✒️ Add new tasks
✒️ Remove existing tasks
✒️ Mark tasks as completed
✒️ Responsive design

Technologies Used

👩🏻‍💻 HTML5
👩🏻‍💻 CSS3
👩🏻‍💻 JavaScript
